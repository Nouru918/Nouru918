# 게임 QA 포트폴리오 - 트릭컬: 리바이브

'트릭컬: 리바이브' 라이브 빌드를 기반으로 수행한 QA 프로젝트 공간입니다.  
본 페이지에서는 버전별 포트폴리오 원본(PDF) 및 분석 데이터 시트(Excel)를 제공합니다.

---

## 1. 테스트 환경 (Test Environment)
* **OS**: Android 06 (Galaxy Tab S9 FE)
* **Build**: 라이브 서비스 최신 빌드 기준
* **Tool**: JIRA (결함 추적 및 워크플로우 관리), Excel (데이터 매트릭스 분석)

## 3. JIRA 결함 관리 요약 (Defect Management)
* **핵심 성과**: 누적 191시간의 장기 플레이 경험을 바탕으로 서비스 리스크 발굴
* **프로세스**: 4단계 결함 관리 워크플로우(진행 대기 → 수정 중 → QA 확인 → 완료)를 준수하여 JIRA 상에 결함 데이터 구축
* **분역별 이슈**: UI 편의성 부재, 연출 로딩 시 암전 프리징, 특정 속성 소외 밸런스 등급 가치 훼손 결함 식별 (상세 내역은 v2.0 PDF 문서의 #3, #4 장표 참고)
